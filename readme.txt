=== ProductPage ===

ProductPage is modern style free eccomerce Landing product page with responsive WordPress theme. It is SEO friendly with featured image supports and custom colors.
This theme is fully widgetized, so it is easier to operate.Theme customizer can be used to change color, upload logo and other theme settings. You can get quick SUPPORT through support page https://99colorthemes.com/support/ and view live DEMO at http://demo.99colorthemes.com/productpage/
